# Social Feed Test

**Social Feed Test** is a simple Flutter test app for posting and viewing posts and comments. Frontend-only, designed for UI practice and experimenting with post interactions.

## Features

- View a list of posts  
- View post details and comments  
- Add new posts  
- Add comments to posts  
- Edit posts  
- Delete posts (with confirmation dialog)  
- Frontend-only: no backend integration  

## Getting Started

This project is built with Flutter and demonstrates frontend UI design and interaction handling.

### Prerequisites

- Flutter SDK installed  
- Dart SDK  
- IDE: **VS Code** or **Android Studio**  

### Installation

1. Clone the repository and navigate into the project directory, then install dependencies and run the project:
