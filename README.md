# Social Feed Test

A simple Flutter test app for posting and viewing posts and comments. Frontend-only, designed for UI practice and experimenting with post interactions.

---

## Features

- View a list of posts
- View post details and comments
- Add new posts
- Add comments to posts
- Edit posts
- Delete posts (with confirmation dialog)
- Frontend-only: no backend integration

---
